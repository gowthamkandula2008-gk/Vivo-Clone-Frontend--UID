# Vivo Website Frontend

A modern and responsive frontend recreation of the Vivo official website. This project is developed as part of the User Interface Design (UID) subject to demonstrate frontend development skills using HTML, CSS, and JavaScript.

---

# ✨ Live Demo

### View Live https://resplendent-blini-2d23eb.netlify.app/



* ⚠️ Best viewed on Desktop / Laptop for better experience.
* 📱 Responsive design works on Mobile, Tablet, and Desktop.
* 🔗 Social media buttons open Vivo official pages.

# 📥 Getting Started

## 1. Clone the Repository

```bash
git clone [https://github.com/yourusername/vivo-website.git](https://github.com/gowthamkandula2008-gk/Vivo-Clone-Frontend--UID)
```

## 2. Open Project Folder

```bash
cd vivo-clone
```

## 3. Run the Website

Simply open `index.html` in your browser.

OR

Use VS Code Live Server extension for better experience.
